# Color Selecting App

## Description
A simple desktop application built using **Visual Studio** and Visual Programming.  
This app allows users to **select, mix, and view colors** interactively.  
It demonstrates GUI design, event-driven programming, and basic user interaction.

## Tech / Tools Used
- Visual Studio (WinForms / WPF)
- C# / VB.NET
- GUI Components: Buttons, Sliders, Panels
- Optional: Laragon (if database connection is used)

## Features
- Select colors using sliders or color picker  
- Mix two or more colors to create custom shades  
- Display RGB and Hex values of the selected color  
- Interactive GUI with responsive controls
